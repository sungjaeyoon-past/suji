## 입지 요약

| 항목 | 내용 | 수지 입지기준 |
|---|---|---|
| **직장** | 강남 직장 의존도 높은 베드타운 | |
| **교통** | a) 강남/판교 직장과 연결되는 신분당선역 접근성이 가장 중요 / b) 영통/기흥 직장과 연결되는 수인분당선역 접근성은 다음 순위로 고려 | 신분당선 역세권 여부 / 수인분당선 역세권 여부 |
| **학군** | a) 이현중/정평중 가는 초등배정이 가장 중요 / b) 초품아+학원가를 이용할 수 있는 단지인지 | 초품아/초중고 / 학원가 |
| **환경** | 편의시설 + 백화점/대형마트 + 연식의 힘 / 3가지 모두 합해졌을 때 압도적 힘 발휘 | 상권 / 연식 |
| **공급** | 더 하위 생활권에 27,28년 공급이 예정되어 있음 | |
